# 📚 Online Library Website – Backend Django 

Backend repository for an Online Library Website developed using Django. The project handles database management, authentication, routing, and core library functionalities for managing books and user interactions.

🔗 Frontend Repository: [Online Library Frontend](https://github.com/Nour131/Online_Library_Website)

---

## Project Overview

This project implements the backend logic of an online library management system using Django. It provides functionalities for managing books, handling user authentication, processing requests, and interacting with the database through Django ORM.

The system is designed to support a complete digital library workflow with organized backend architecture and scalable functionality.

---

## Features

- User authentication and authorization
- Book management system
- Database integration using Django ORM
- Add, update, and delete books
- Search and filtering functionality
- Backend routing and request handling
- Django admin panel support
- Structured and scalable backend architecture

---

## Technologies Used

- Python
- Django
- SQLite
- Django ORM

---

## Backend Functionalities

### Authentication System
- User registration
- Login and logout
- Session management

### Library Management
- Add books
- Update book information
- Delete books
- Manage book records

### Database Operations
- Store and retrieve library data
- Handle user and book models
- Database migrations using Django ORM

---

## Project Structure

The backend follows Django’s MVT architecture and includes:

- Models
- Views
- URL routing
- Database configuration
- Authentication system
- Admin panel configuration
